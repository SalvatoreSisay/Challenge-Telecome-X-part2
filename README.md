# 📊 Telecom X - Modelo Predictivo de Churn

Análisis exploratorio de datos, ETL y construcción de un modelo predictivo para detectar la evasión de clientes en Telecom X. Esta es la **Parte 2** del proyecto, enfocada en modelado predictivo.

## 🎯 Objetivos

- Identificar patrones de comportamiento de clientes en riesgo de abandono
- Desarrollar un modelo predictivo robusto para anticipar churn
- Proporcionar insights accionables para retención de clientes

## 📁 Estructura del Proyecto

```
.
├── Telecom_X_parte2.ipynb       # Notebook principal con EDA, ETL y modelos
├── telecom_datos_tratado.csv    # Dataset procesado listo para análisis
└── README.md
```

## 🛠️ Tecnologías Utilizadas

- **Python 3.x**
- **Pandas** - Manipulación y análisis de datos
- **Scikit-learn** - Modelado predictivo
- **Matplotlib / Seaborn** - Visualización de datos

## 📊 Dataset

- **Fuente:** Telecom X
- **Tamaño:** Dataset procesado con características relevantes para predicción de churn
- **Variables clave:** Comportamiento de clientes, uso de servicios, datos demográficos

## 🚀 Cómo Usar

1. Clona este repositorio
2. Abre `Telecom_X_parte2.ipynb` en Jupyter Notebook
3. Ejecuta las celdas secuencialmente para reproducir el análisis

## 📈 Resultados Esperados

El notebook incluye:
- Análisis exploratorio completo del dataset
- Ingeniería de características
- Entrenamiento y evaluación de múltiples modelos
- Métricas de rendimiento y recomendaciones
